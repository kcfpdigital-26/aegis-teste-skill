# Aegis Delivery OS - Framework AI-First

Página de apresentação do framework AI-First Delivery OS, focada em unir a velocidade de execução da IA com a governança da engenharia de software tradicional.

## Tecnologias
- React
- Tailwind CSS
- Vite
- TypeScript

## Estrutura
- **Hero**: Proposta de valor principal.
- **3 Camadas**: Software, Projetos e Processos.
- **Módulos**: Destaque para o Módulo M2 (Auditoria e Recuperação).
